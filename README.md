# Meu RH

Sistema de RH completo em nuvem para pequenas e médias empresas brasileiras.

## Funcionalidades

- Gestão de Colaboradores
- Controle Financeiro de Folha
- Benefícios Flexíveis
- Portal do Colaborador
- Gestão Multi-empresa
- Relatórios e Contabilidade
- Contracheques Digitais
- Controle de Férias

## Tecnologias

- React
- TypeScript
- Tailwind CSS
- Vite
- Supabase

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```
